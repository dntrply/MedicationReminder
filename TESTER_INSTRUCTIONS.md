# Medication Reminder App - Tester Instructions

## Important: Android Only
**This app only supports Android phones.** You must have an Android device to participate in testing.

---

## For New Testers (First Time Users)

### What is this app?
Medication Reminder is a simple, elderly-friendly app that helps you remember to take your medications on time. Features include:
- **Visual medication management** with photos
- **Multi-language support** (English, Hindi, Gujarati, Marathi, Hinglish)
- **Daily timeline view** showing when to take each medication
- **Smart notifications** with encouraging messages
- **Reports and tracking** to monitor your medication adherence
- **Audio notes** to record special instructions

### Installation Steps

1. **Download the APK file** (you'll receive this separately)
   - File name: `app-release.apk`
   - File size: ~28 MB

2. **Enable installation from unknown sources**:
   - Go to Settings → Security (or Apps)
   - Enable "Install unknown apps" or "Unknown sources"
   - Allow installation from your browser/file manager

3. **Install the app**:
   - Open the downloaded APK file
   - Tap "Install"
   - Wait for installation to complete
   - Tap "Open"

4. **Start using the app**:
   - Select your preferred language
   - Add your first medication using the + button
   - Take a photo of the medication
   - Set reminder times
   - Done! You'll receive notifications at scheduled times

## For Existing Testers (Upgrading from v0.11.0)

### What's New in v0.17.2?
Since v0.11.0, we've added major new features:
- **Reports screen** with adherence tracking, calendar view, trends graph, and per-medication statistics
- **Gamified notifications** with encouraging messages and streak tracking
- **Hinglish language** support (mix of Hindi and English)
- **Multi-profile support** for shared devices
- **Audio transcription** (optional) to convert voice notes to text
- **Improved timeline view** with better medication stacking and visual indicators
- **Bug fixes** for notification behavior, missed dose tracking, and UI improvements

### Installation Steps - **IMPORTANT: Must Uninstall First**

**⚠️ WARNING: You must uninstall the old version first. You will need to recreate all your medications and data.**

1. **Uninstall the current version**:
   - Go to Settings → Apps
   - Find "Medication Reminder"
   - Tap "Uninstall"
   - Confirm uninstallation
   - **Note:** This will delete all your current medications and history

2. **Download the new APK file** (you'll receive this separately)
   - File name: `app-release.apk`
   - File size: ~28 MB

3. **Install the new version**:
   - Open the downloaded APK file
   - Tap "Install"
   - Wait for installation to complete
   - Tap "Open"

4. **Recreate your data**:
   - Select your preferred language
   - Add your medications again using the + button
   - Set up reminder times
   - Explore the new Reports screen (accessible from the menu)

## Need Help?

Common questions:
- **Notifications not working?** Check that battery optimization is disabled for the app (Settings → Apps → Medication Reminder → Battery → Unrestricted)
- **App closes when pressing back button?** This has been fixed in v0.15.1+
- **Timeline view confusing?** Try the Reports screen for a calendar-based view
- **Want to export data?** Go to Settings → Reports → Export Data

Thank you for helping test the Medication Reminder app!
